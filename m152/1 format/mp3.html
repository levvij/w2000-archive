File format mp3
Full name MPEG Audio Layer III
Media type

Information loss
Description

MP3 is the most commonly used file format for Audio. It is compressed by removing details most humans can't hear and audio systems can't play, but saves a lot of space

Advantages
  • Small
  • Can contain ID3 tags
  • Plays on almost every device
  • Can be live streamed
Disadvantages
  • Lossy
  • Requires a decoder
Usage in web

Can be used with the standard audio tag (type = mpeg)
